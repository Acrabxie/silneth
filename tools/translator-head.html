当前支持的句型（规则直译是诚实的：不支持的会明说）
  • 主谓（宾）句，含时间词定级：现在/昨天/以前/远古/明天…
  • 示证词：亲眼/据说/相传/看来…（级1、级2 陈述句会自动补示证）
  • 系词句「X是Y」· 处所句「X在Y」· 领有句「X有Y」
  • 疑问：什么/谁/哪里/何时/几 + 是非问 · 否定「不/没/别」
  • 祈使与意愿「…吧」 · 比较「X比Y…」 · 属格「X的Y」 · 复数/全称量化
  • 被动「X被(Y)…」：受事升主格，施事取工具格 -um，动词加中受动 -u-
  • 名词并列「A和B / A或B」 · 限定词「所有/每/某/任何/许多/一些/几个＋名词」
  • 复句：如果…那么 / 因为…所以 / 虽然…但是 / 而且 / 除非 / 为了（连词或逗号切分，各从句独立定级）
  • 动词补语「想要唱 / 需要走」：补语动词取副动词 -um，矩阵动词变位
  • Subject–verb–(object) with time words setting the tense level: now / yesterday / before / in ancient times / tomorrow…
  • Evidentials firsthand / reportedly / apparently… (level-1 & level-2 statements auto-add one)
  • Copula “X is Y” · locative “X is in Y” · possession “X has Y”
  • Questions what / who / where / when / how many + yes-no · negation not / never · imperative “sing!”
  • Comparison “X is …-er than Y” · possessive “your name” · plural / regular & irregular inflection
  • Passive “X was broken (by Y)”: patient→NOM, agent→instrumental -um, verb takes middle -u-
  • Coordination “A and/or B” · determiners all / each / every / any / many / some / several / a certain
  • Clause linking if…then / because / so / although / but / unless / moreover — connective or comma splits, each clause tensed independently
  • Verb complements “want to sing / need to go”: complement verb takes converb -um, matrix verb conjugates
外来之名以满元音收录——新来的词，尚未被时间磨蚀。
