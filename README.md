# Sìlneth /ˈsɪl.neθ/ · 承继而来的话语

一门以**元音磨蚀**为发动机的人造语言，以及一个把主流语言译入它的**翻译器**。

所言之事离现在越远，动词词根的元音就被磨得越平——说今天的事元音饱满，说上一代的事元音松弛，说祖先与神话之时元音坍缩成一声气化的 `ë`。「古老」不是被标注的，而是被直接听见的。

## 在线

- **翻译器** — [`index.html`](./index.html)：中文 / 英文直译（含十步翻译管线过程）、构句器、外来名音译、303 词根词典、常用语，全部实时渲染 Mírneth 原生文字。
- **语言介绍** — [`language.html`](./language.html)：设计哲学、音系、文字系统、语法、例文、歌者之民。

## 招牌特征

| 域 | 机制 |
|---|---|
| 时态 | 词根元音三级磨蚀：级0 近时 / 级1 远时 / 级2 祖时 |
| 示证 | 句末附着词，元音随时态同步磨蚀，传闻可迭代计手数 |
| 修辞 | 恒守体（弱根拒磨＝誓言）／倾颓体（强根强制坍缩＝挽歌） |
| 文字 | Mírneth：长元音架天际线、级1 刻裂痕、级2 磨平为地平线 |

## 引擎

`index.html` 内联了一个纯 JS 翻译引擎（`SilnethEngine`）：磨蚀级词形推导、R1 滑音 / R2 插音连读、名词变格、动词模板、示证同步律、罗马字→Mírneth 字形转写、中英文规则直译与外来名音译。引擎经 127 项 node 测试（对照形态规格的 86 个书证向量）全绿。

这是一门规则自洽的语言：**给一个词根，规则足以机械推出它的全部形式。**
